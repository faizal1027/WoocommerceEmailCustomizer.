WooCommerce Email Template Customizer lets you design beautiful, branded transactional emails for WooCommerce while integrating Mailchimp to send targeted campaigns and sync customer data effortlessly.

== Description ==

**WooCommerce Email Template Customizer** is a powerful plugin that helps WooCommerce store owners personalize their customer email communication. With a drag-and-drop editor and deep Mailchimp integration, you can craft stunning emails, automate marketing, and increase conversions—all from your WordPress dashboard.

**Key Features:**

- **Drag & Drop Email Builder**  
  Visually customize order receipts, customer notifications, password resets, and more—no code required.

- **Mailchimp Sync Integration**  
  Automatically push customer and order data to Mailchimp for segmented targeting and automation workflows.

- **Custom Email Templates**  
  Modify default WooCommerce email templates with your branding, layout, and style.

- **Advanced Conditions & Placeholders**  
  Use dynamic content and conditional logic to personalize messages per user or order type.

- **Developer-Friendly Hooks**  
  Customize with filters, actions, and template overrides. Great for agencies and freelancers.

**Smarter than default WooCommerce emails:**  
Forget boring plain-text emails—this plugin brings modern, responsive, and marketing-ready emails to your WooCommerce store.

== Installation ==

1. Upload the plugin to your `/wp-content/plugins/` directory or install it via the WordPress dashboard.
2. Activate the plugin through the "Plugins" menu.
3. Navigate to **WooCommerce → Email Customizer** to start building your email templates.
4. Connect Mailchimp under **Settings → Mailchimp**.

== Frequently Asked Questions ==

= Can I edit default WooCommerce emails? =  
Yes. You can visually edit all transactional emails using the built-in email builder.

= Is Mailchimp required to use this plugin? =  
No. You can use the customizer independently. Mailchimp sync is optional.

= Will this work with other WooCommerce plugins? =  
Yes. It’s compatible with most WooCommerce extensions and themes.

= Can I create different emails based on user roles or order types? =  
Yes. The plugin supports conditional display and smart tags for personalization.
